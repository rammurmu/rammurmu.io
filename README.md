## Setup Instructions
1. Clone the repository:
    ```bash
    git clone https://github.com/rammurmu/rammurmu.io.git
    ```
2. Navigate to the project directory:
    ```bash
    cd rammurmu.io
    ```
3. Open `index.html` in your browser to view the website.
